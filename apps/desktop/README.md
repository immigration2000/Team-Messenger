# Desktop App (Electron + React)

## Requirements
- Node.js 24+
- Windows PowerShell에서는 `npm` 대신 `npm.cmd` 사용

## Install
```bash
npm.cmd install
```

## Run (Dev)
```bash
npm.cmd run dev
```

## Build
```bash
npm.cmd run build
```

## Verified
- `npm.cmd install`: success
- `npm.cmd run build`: success
- `dev:renderer` (Vite): port `5173` listening 확인
- Electron binary: `v31.7.7`
