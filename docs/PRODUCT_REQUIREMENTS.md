# Product Requirements (MVP)

## 핵심 사용자
- 팀원
- 팀 리더/매니저
- 관리자

## 핵심 기능
1. 메신저
- 1:1 채팅, 그룹 채팅
- 파일 첨부
- 읽음 상태

2. 일정 관리
- 프로젝트 캘린더
- 마일스톤/업무 일정 등록
- 알림

3. 휴가 관리
- 휴가 신청
- 승인/반려
- 팀 캘린더 반영

4. 관리자 기능
- 조직/팀 관리
- 권한 관리

## MVP 범위
- 이메일 로그인
- 팀 단위 채팅
- 월간 캘린더
- 휴가 신청/승인
- 푸시/인앱 알림

## 비기능 요구사항
- 메시지 송수신 지연 최소화
- 모바일/데스크톱 일관된 UX
- 감사 로그(휴가 승인 이력)
